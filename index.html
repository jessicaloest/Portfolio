XING Suche

XING alle Plattformen

XING Suche mit einer Person

Die XING Suche sollte im UI bereich verbessert werden. Die Suche sollte mit einem echten User nachgebaut werden. Dabei sollen alle verschiedenen Unterkategorien auf einer Seite angezeigt werden. Und nicht mehr auf verschienden Tabs. "Top Suchergebnisse" und "Weitere Suchergebnisse" sollen die einzigen Kategorien sein.

Notification Center

Empty State

Das Notification Center (Mitteilungsbereich) ist eine React-Applikation auf der XING Webseite. Ein Empty State sollte in der Notifications Center APP zu sehen sein, wenn der Nutzer noch keine Mitteilungen hat. Um den Empty State darzustellen habe ich eine React Komponente gebaut, die aus einem Bild, einer Headline, einem Text und einen Button besteht.

System E-Mails

Neue Information

Die System E-Mails werden XING als HTML Template mit dynamischen Daten in eine Rails-Applikation eingebettet. Dort habe ich habe bei einigen E-Mails neue Information zum Beispiel bei einem anstehenden Event hinzugefügt. In dieser E-Mail sollte der Eventveranstalter, freie Plätze und der Eventort hinzugefügt werden, damit der Nuzer noch mehr über das Event weiß. Im Fall, dass eine Information in dem Event nicht vorhanden ist, wird ein Ersatz oder eines leeres Feld angezeigt.

System E-Mails

Neues Design

Die E-Mails werden zur Zeit in dem neuen XING Design umgebaut. Dabei habe ich mitgeholfen. In den neuen E-Mails werden Komponenten zusammengefügt und mit unechten Daten gefüllt. Dabei werden schon vorhandene Komponenten verwendet oder neu erstellt.Jede neu erstellte E-Mail wird sowohl als HTML Template Variante und Textvariante erstellt.

System E-Mails

Dark Mode

Nach einiger Recherche, habe ich eine Liste der Clients erstellt, die Dark Mode unterstützt und welche nicht. Dadurch konnten wir besser verstehen wo der Dark Mode funktionieren wird und wo nicht.
